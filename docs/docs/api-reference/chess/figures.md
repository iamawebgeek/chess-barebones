# Figures

## `Figure` Enum

An enum of all the chess pieces:

-   `KING`
-   `QUEEN`
-   `ROOK`
-   `BISHOP`
-   `KNIGHT`
-   `PAWN`

## Figure Classes

-   `King`
-   `Queen`
-   `Rook`
-   `Bishop`
-   `Knight`
-   `Pawn`

## `ChessFigureFactory`

A `FigureFactory` that creates decorated chess figures.
